Asset Tracker - Personal Finance Manager

Asset Tracker is a full-stack "MERN" (MongoDB, Express, React, Node) application designed to help users track their savings and visualize financial goals. It features a sleek, secure dashboard with real-time progress tracking.

---

Live Demo
Check out the live application here:  
**(https://asset-tracker-ruby-iota.vercel.app/)**

---

Key Features
Secure Access:** Integrated an "Access Key" protection system to secure financial data.
Guest Access (Demo Mode):** Added a specialized "Demo Mode" for recruiters to explore the dashboard without needing a password.
Dynamic Progress Tracking:** Real-time progress bar that calculates the gap between current savings and the defined financial target.
Ledger Management:** Full CRUD functionality to log revenue sources and delete entries.
Responsive Dark UI:** A modern, mobile-friendly interface built with custom CSS and a focus on User Experience (UX).
